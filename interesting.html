This is a collection of interesting articles and research papers to read. It is not limited to science or mathematics.

  • Title: Advancing mathematics by guiding human intuition with AI site

    Authors : Alex Davies, Petar Veličković, Lars Buesing, Sam Blackwell, Daniel Zheng, Nenad Tomašev, Richard Tanburn, Peter Battaglia, Charles Blundell, András Juhász, Marc Lackenby, Geordie Williamson, Demis Hassabis & Pushmeet Kohli

    Short Description : Using AI to extract new insights in topology. A very hard read as well but still interesting.

  • Title: The Most Famous Paradox in Physics Nears Its End site

    Author : George Musser

    Short Description : A relatively hard read on the Black Hole Information Paradox but it is still a good read.

  • Title: Crystalball: Gazing in the Black Box of SAT Solving paper

    Authors : Mate Soos, Raghav Kulkarni, Kuldeep S. Meel

    Short Description : Using Machine Learning on SAT Solvers to find useful lemmas.

  • Title: Bosphorus : Bridging ANF and CNF Solvers paper

    Authors : Davin Choo , Mate Soos , Kian Ming A. Chai , and Kuldeep S. Meel

    Short Description : Solving ANF faster using CNF solvers using the Bosphorus mastersolver

  • Title: Mathematicians Prove 2D Version of Quantum Gravity Really Works site

    Author : Charlie Wood

    Short Description : Louville Quantum Field Theory which presents a 2D model of quantum gravity. Another hard read

  • Title : FermiNet : Quantum Physics and Chemistry from First Principles Blog Post

    Authors : Authors : David Pfau, James Spencer, Alexander Matthews, Matthew Foulkes

    Short Description : Using Deep Learning in helping us compute the energy of atoms and molecules from first principles which can help to solve the fundamental equations for Quantum Mechanics for real-world systems.

  • Title : Game theory as an engine for large-scale data analysis Blog Post

    Authors : Brian Mcwilliams, Ian Gemp, Claire Vernade

    Short Description : This blog shows us a very novel application of Principal Component analysis (PCA) as a multi-agent game called EigenGame.

  • Title : The Implementation of Partial Least Squares with Artificial Neural Network Architecture Paper

    Authors : Tzu-Chien Hsiao, Chii-Wann Lin, Mang-Ting Zeng, H.-H.K. Chiang

    Short Description : This paper demonstrtates how partial least squares can contribute weights to initialise the weights and bias(?) of the neural network

  • Title : Attention is all you need Paper

    Authors : Ashish Vaswani, Noam Shazeer, Niki Parmar, Jakob Uszkoreit, Llion Jones, Aidan N. Gomez, Lukasz Kaiser, Illia Polosukhin

    Short Description : Introduction to the new state of the art transformer architecture for deep learning

  • Title : Extending SAT_Solvers to Cryptographic Problems Paper

    Authors : Mate Soos, Karsten Nohl, and Claude Castelluccia

    Short Description : SAT Solvers which are tweaked to perform solve cryptographic functions

  • Probabilistic Deep Learning with Probabilistic Neural Networks and Deep Probabilistic Models Paper

    Authors : Daniel T. Chang

    Short Description : About deep neural networks that utilize probabilistic layers which can represent and process uncertainty.

    * Introduces Tensorflow Probability Library API
